2.安装步骤

①解压Maven文件：

这里我解压到的位置是：D:\develop（注意：目录不要带空格）

②配置本地仓库：

在你的maven目录中新建一个mvn_repo(指定为Maven的本地仓库)

然后修改maven的配置文件，打开conf文件中的settings.xml(打开方式不要选记事本，此处我选择的是vs code)，然后找到D:\develop.m2\apache-maven-3.9.11\mvn_repo

将其复制粘贴到注释之外，接着往下拉，找到mirror标签(不是mirrors)，将原本的内容删除，配置阿里云的私服地址：

​ alimaven

​ aliyun maven

​ http://maven.aliyun.com/nexus/content/groups/public/

​ central

③配置环境变量

打开系统属性 -> 环境变量 -> 新建 -> 变量名：MAVEN_HOME，值为你所下载的maven的位置，接着在PATH路径中添加%MAVEN_HOME%\bin

全部点击确定以后打开cmd输入mvn -v,若出现以下样式，则说明安装成功了

IDEA集成Maven项目

①创建Maven项目

完成以上全局配置(注意：是全局配置)：首先点击close-project，再点击Customizs，接着点击All settings即可进入到全局配置

创建模块，选择new module，填写模块信息，选择构建工具为Maven，点击create，创建完成

②Maven坐标

概念：是资源(jar)的唯一标识，通过该坐标可以唯一定位资源位置

使用坐标来定义项目或引入项目中需要的依赖

主要组成：

  • groupId：定义当前Maven项目隶属组织名称(通常是域名反写)
  • artifacId:定义当前Maven项目名称(通常是模块名称)
  • version:定义当前版本号
    • SNAPSHOT：功能不稳定、尚处于开发中的版本，即快照版本
    • RELEASE：功能趋于稳定、当前更新停止，可以用于发行的版本

③导入Maven项目

方式一：File -> Project Structure -> Modules -> Import Module -> 选择maven项目的pom.xml
